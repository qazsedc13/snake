# Snake

## Описание
Данный проект представляет собой классическую консольную игру "Змейка", реализованную на языке программирования C#. На текущем этапе проект содержит базовую структуру консольного приложения и готов к дальнейшему расширению игровой логики.

## Технологический стек
- **Язык программирования:** C#
- **Платформа:** .NET Framework 4.5.2
- **Тип приложения:** Console Application (Консольное приложение)
- **Среда разработки:** Microsoft Visual Studio

## Структура проекта
- `snake/` — основная папка решения (Solution).
  - `snake.sln` — файл решения Visual Studio.
  - `snake/` — папка проекта с исходным кодом.
    - `Program.cs` — точка входа в приложение (Main).
    - `snake.csproj` — файл конфигурации проекта.
    - `App.config` — конфигурационный файл приложения.
    - `Properties/AssemblyInfo.cs` — метаданные сборки.

## Установка и запуск
### Требования
- Установленная среда разработки **Microsoft Visual Studio** (версии 2015 или выше).
- Установленный пакет **.NET Framework 4.5.2 Developer Pack**.

### Инструкция по запуску
1. Склонируйте репозиторий:
   ```bash
   git clone git@github.com:qazsedc13/snake.git
   ```
2. Откройте файл решения `snake/snake.sln` через Visual Studio.
3. Дождитесь загрузки проекта и восстановления зависимостей.
4. Нажмите кнопку **"Start" (Пуск)** или клавишу `F5` для компиляции и запуска приложения.

## Основные функции
На данный момент приложение выполняет следующие действия:
- Инициализация консольного окна.
- Вывод приветственного сообщения "Hello World!!!".
- Ожидание ввода пользователя перед завершением работы.

*Примечание: Игровая логика (перемещение змейки, сбор еды, обработка столкновений) находится в процессе разработки.*

## Пример использования
После запуска приложения в консоли появится сообщение:
```text
Hello World!!!
```
Для завершения работы программы нажмите любую клавишу.
